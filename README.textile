This is *jQuery* autocomplete plugin

*Usage example*

<pre>
var options = {
	varName: 'input',
	requestType: 'GET',
	
	url: siteURL + "/users/ajax_search?",
	
	/**
	 * var item - clicked item (html objected extended by jQuery)
	 * var obj - clicked object (initial object used for rendering of autocomplete items) 
	 */
	selected: function(item, obj)
	{
		var userContainer = $('#user_' + obj.id);
		if (!userContainer.length) {
			$('#userListContainer').append(obj.userHtml);
			$('#users').val('');
		}
	},
	
	/**
	 * var @item - object from response 
	 * returns value used for rendering of each autocomplete item 
	 */
	parseItem: function(item)
	{
		return item.value;
	}
};

$("#users").autocomplete(options);
</pre>

In order to get it work you have to response *array of objects* in *json* encoded string.
